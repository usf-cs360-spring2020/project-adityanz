Alpha Release

  • Website Requirements:

    • Home/Landing Page: the dedicated page that describes the overall narrative and provides clear navigation to the other pages.

    • About Page: A separate page that describes you and your motivation behind this project.

    • Dataset Page: A separate page that describes the original dataset. includes: the dataset attribution, the license for the dataset, the size of the dataset, the content of the dataset

    • Prototypes Page or Section: A separate page (or pages) that will include your various prototypes and feedback you received on those prototypes.

  • Prototype Requirements:

    • Create a simple prototype of your dataset

Beta Release

  • Website Requirements:

    • Refined Theme: A more refined and clearly marked (ideally using a section heading) section the Home/Landing Page with the overall narrative or theme for your project.

    • Prototype Page:A new prototype page (the alpha prototype should be on a separate page) for the beta prototype. This page should include the prototype, and the following clearly marked writeups: Question,Answer,Encoding ,Interactivity

  • Prototype Requirements:

    • The visualization prototype may be created using any tool and does not have to be interactive. However, it does have to be one of the visualization techniques listed for your primary final project visualization.
